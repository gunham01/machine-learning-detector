### Yêu cầu hệ thống
- Python 3.11.4
- Hệ quản trị CSDL MySQL (phiên bản mới nhất)


### Các thư viện dùng trong dự án:

-   scikit-learn
-   mysql-connector-python
-   pandas
-   env

Trước khi chạy dự án, hãy install từng thư viện bằng lệnh

```bash
pip install <tên thư viện>
```

VD:

```bash
pip install scikit-learn
```
